# Backend For TodoApp

- First to create package.json file , npm init
- To set up backend required modules, npm i express mongoose cors nodemon
- Set up .gitignore file with necessary lines , use gitignore generator
- Create index.js file , inside src folder(optional)
- In package.json under scripts section code => "start": "nodemon index.js"

## How to run Backend
- npm start command in terminal
